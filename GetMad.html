Projekt Übersicht

GetMad ist ein next Level AntiCheat Plugin für Spigot. Es zeichnet sich vorallem durch seine hervorragenden Scaffold, Fly, sowie Speed Checks aus. Alle anderen Checks sind jedoch nicht außer acht zu lassen... Durch seine hohe Präzession sind Fehlbanns so gut wie ausgeschlossen.

Features

- Logging von Flags
- Durch die Configuration File vollständig konfigurierbar.
- Konfigurieren von Checks
- Webhook für Flags und Aktivitätsstatus
- Banneffekte
- Coming soon: AdminControlPanel für einfache Verwaltung und erweiterte Funktionen.
- Coming soon: Auto-Banns mit WaterBan Plugin verknüpfen
